Система бронирования отеля
Полнофункциональная асинхронная веб-система для автоматизации бронирования номеров в отеле. Реализует полный цикл работы с сущностями:
пользователи, номера, бронирования. Для всех сущностей доступны CRUD-операции через REST API.

Функции приложения:
Регистрация или удаление пользователей в базе
Искать доступные номера по датам и уровню комфорта
Бронировать номера на выбранные даты
Просматривать историю бронирований пользователя
Управлять пользователями, номерами, бронированиями
Просматривать все бронирования отеля

Техническая реализация
Полностью асинхронная архитектура (async/await)
Интеграционные тесты для всех API-эндпоинтов
Валидация данных и обработка ошибок
Оптимизированная работа с базой данных
